meshwidget
==========

Make [trimesh2][] simpler in Qt project

Features
--------

*	Cross-platform
*	So simpler

Usages
------

1.		```C++
		MeshWindow meshWindow;
    	meshWindow.showMaximized();
		```

2.		```C++
		MeshWidget mw;
    	mw.readObj("data/bunny.obj");
    	mw.showMaximized();
		```
